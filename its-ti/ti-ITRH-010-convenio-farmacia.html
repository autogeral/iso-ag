1. Objetivo

O objetivo desta instrução de trabalho é orientar sobre o procedimento para a geração de arquivos de Convênio Farmácia para lançamento por importação de arquivos no Sage, diminuindo o trabalho manual.

2. Referências

O repositório para o projeto Folha está disponível em: https://source.cloud.google.com/folha-s/folha

3. Definições

4. Procedimentos e Responsabilidades

4.1. Clonar o projeto

4.2. Criar o banco de dados folha na máquina local

4.3. Cadastrar as unidades Auto Geral se necessário

Cadastros > Unidades organizacionais

4.4. Solicitar à Infra a liberação da na porta do servidor com a máquina do Sage

4.5. Criar conexão com o banco postgresql, solicitar as credenciais com o supervisor do TI

4.6. A conexão é criada em FuncionarioSageService

4.7. Rodar o projeto, ele será inicializado na porta 8090

4.8. Enviar os arquivos de retorno, é possível realizar o upload de vários arquivos

4.9. Selecionar o período que irá gerar o arquivo

4.10. Será criado o arquivo Zip, importar cada um a sua respectiva loja no Sage
