1. Objetivo

O objetivo desta Instrução de Trabalho é estabelecer a sistemática adotada para conferência do fluxo de caixa.

2. Referências

Não aplicável

3. Definições

Não aplicável

4. Procedimentos e Responsabilidades

4.1 Recebimentos dos malotes que são enviados pelas lojas do dia anterior

-Acessar o Manual de usuário de Conferência de Caixas no GLPI, através do Link: clique aqui

-O responsável faz a conferência dos valores x relatório.
-Lançar no sistema;
-Conferir se vieram todas as diárias;
-Organizar os dinheiros;
-Separar os cheques à vista e cheque-pré;
-Conferir se os valores lançados batem no ERPJ;
-Conferir o relatório do cheque-pré (do dia);
-Alterar no ERPJ, a situação dos cheques para (3) depositado;
-Carimbar e conferir os dados de todos os cheques;
-Fazer a sangria no ERPJ e enviar para depositar em banco;
-Conferir sangrias enviadas pelos caixas com o extrato da conta corrente;
-Conferir o caixa;
-Arquivar os cheques-pré na “caixa”;
-Entregar os fechamentos de caixa para conferência detalhada.

4.2 Conferência detalhada

-Relatórios de pagamentos (Conferir título X diária);
-Conferência das vias de pagamentos de cartões;
-Despesas e centro de custo;
-Fechamento geral dos caixas (Relatório de cheques, SAT, troco);
-Após a conferência de todos os itens os relatórios e fechamento de caixa, são arquivados ;

4.3 Conferência extratos bancários

-Tirar os extratos de todas as contas bancárias;
-Conferir sempre o saldo do dia anterior com o atual;
-Conferir as entradas(Boleto/Sat/NF,nome);
-Conferir se os valores dos destinos (Boleto/Sat/NF) batem com o extrato;
-Checar as transferências entre contas da empresa.
-Fazer a Transferência de saldo do Banco Inter para o Santander;
